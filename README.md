# Lumiko Final Pack

Скачивание без багов ChatGPT:

1. Открой PowerShell.
2. Скачай репозиторий ZIP:

```powershell
Invoke-WebRequest -Uri "https://github.com/alexe44492-rgb/Eat/archive/refs/heads/main.zip" -OutFile "$env:USERPROFILE\Desktop\lumiko_github_pack.zip"
```

3. Распакуй `lumiko_github_pack.zip`.
4. Зайди в распакованную папку `Eat-main`.
5. ПКМ по `restore_lumiko_zip.ps1` -> Run with PowerShell.

После этого рядом появится нормальный архив:

`lumiko_final_pack.zip`

Внутри него:
- `01_PLUGIN_SOURCE_FIXED`
- `02_COMMANDPANELS_FIXED`
- `03_ITEMSADDER_FIXED_CONTENTS`
